## Getting Started
---
#### Navigate to the home directory 🌍
```
cd ~
```

#### Clone repo 📠
```
git clone https://github.com/msorce/.dotfiles.git
```

#### Navigate to .dotfiles repository 🗃
```
cd .dotfiles/
```

#### Run setup command 🏃
```
./setup.sh
```
